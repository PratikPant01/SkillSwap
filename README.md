# SkillSwap

**SkillSwap** is a student-focused skill exchanging and learning platform designed to help learners **enhance, share, and transform their skills** through collaboration. Instead of traditional paid courses, SkillSwap enables students to **exchange skills with each other**, making learning more accessible, practical, and community-driven.

---
## Tech Stack

- **Frontend:** Next.js, Tailwind CSS  
- **Backend:** Node.js, Express  
- **Database:** PostgreSQL  
- **Authentication:** JWT , OAuth
