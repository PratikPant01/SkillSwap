# SkillSwap

SkillSwap is a peer-to-peer platform designed to help people trade knowledge and skills.

---

## Installation & Setup

### 1 Clone the Repository

````bash
git clone https://github.com/YOUR_USERNAME/skillswap.git
cd skillswap

### 2 frontend
```bash

cd client
npm install
npm start

### 3 Backend

cd server
npm install
npm start
````
